# 🏏 IPL Auction App

## 📌 Overview
This project is a web-based IPL Auction Simulator that replicates the real-world IPL auction process. Users can bid on players, manage team budgets, and create their own squads.

## ✨ Features
- Real-time bidding system
- Budget management for teams
- Dynamic player display
- Interactive UI

## 🛠️ Tech Stack
- HTML, CSS, JavaScript
